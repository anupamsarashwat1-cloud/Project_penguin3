# Frontend Module

This directory contains the RTL sources for the frontend subsystem of the TITAN-X SoC.

## File Descriptions

- **`rv_bpu.v`**: Branch Prediction Unit (BPU) with BTB and BHT.
- **`rv_decode.v`**: Instruction Decode stage of the pipeline.
- **`rv_fetch.v`**: Instruction Fetch stage, issuing requests to the L1I cache.
- **`rv_icache.v`**: L1 Instruction Cache (L1I) controller and arrays.
