# Security Module

This directory contains the RTL sources for the security subsystem of the TITAN-X SoC.

## File Descriptions

- **`drbg.v`**: Deterministic Random Bit Generator (DRBG).
- **`ecdsa_engine.v`**: Elliptic Curve Digital Signature Algorithm (ECDSA) accelerator.
- **`envm_ctrl.v`**: eNVM (Non-Volatile Memory) controller for secure storage.
- **`secure_boot.v`**: Secure Boot ROM and verification logic.
