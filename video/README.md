# Video Module

This directory contains the RTL sources for the video subsystem of the TITAN-X SoC.

## File Descriptions

- **`hdmi_ctrl.v`**: HDMI display controller.
- **`isp_pipeline.v`**: Image Signal Processor (ISP) camera pipeline.
- **`mipi_csi2_rx.v`**: MIPI CSI-2 receiver for camera input.
- **`vdma.v`**: Video Direct Memory Access (VDMA) engine.
