# Peripherals Module

This directory contains the RTL sources for the peripherals subsystem of the TITAN-X SoC.

## File Descriptions

- **`aes_engine.v`**: AES cryptographic accelerator engine.
- **`can_controller.v`**: CAN bus controller.
- **`gem_ethernet.v`**: Gigabit Ethernet MAC (GEM) controller.
- **`gem_sgmii_pcs.v`**: SGMII PCS for Gigabit Ethernet.
- **`gpio_ctrl.v`**: General Purpose Input/Output (GPIO) controller.
- **`i2c_master.v`**: I2C master controller.
- **`pcie_pipe_if.v`**: PCIe PIPE interface logic.
- **`pcie_top.v`**: PCIe Root Complex/Endpoint top-level module.
- **`rtc.v`**: Real-Time Clock (RTC) module.
- **`sha256_engine.v`**: SHA-256 cryptographic hashing engine.
- **`spi_master.v`**: SPI master controller.
- **`trng.v`**: True Random Number Generator (TRNG).
- **`uart_16550.v`**: 16550-compatible UART controller.
- **`watchdog_timer.v`**: Watchdog timer module.
